Документ «Акт отбора проб номенклатуры»

Документ «Акт отбора проб» служит для отражения факта взятия проб для внутренней или внешней сертификации.

Для отражения списания ТМЦ с целью отбора проб на основании документа «Акт отбора проб» может быть введен документ «Требование-накладная».

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа с помощью кнопки «Операция».

В документе заполняется информация о номенклатуре, серии номенклатуры, складе с которого производится отбор проб. Поле «Нормативный документ» заполняется системой автоматически, на основании данных регистра сведений «Нормативные документы сертификации номенклатуры», но может быть выбрано вручную, из справочника «Нормативные документы сертификации номенклатуры».

Если выбран вид операции «для внутренней сертификации», доступны следующие поля:

На закладке «Пробы для внешней сертификации» - справочные (для печатных форм) поля - «количество образцов, отобранных для испытаний», «количество образцов, отобранных для контрольной пробы». Также указывается единица измерения (единица отбора проб).

При любом виде операции, на закладке «Дополнительно», находятся следующие поля: результат наружного осмотра, замечания к упаковке, маркировке, количество вскрытых упаковок - справочная информация, предназначенная для формирования печатных форм.

Для вида операции «для внутренней сертификации» доступны следующие печатные формы:

- Акт отбора и распределения проб исходных материалов (сырья)

- Акт отбора и распределения проб лекарственного растительного сырья

- Акт отбора и распределения проб упаковочных и маркировочных материалов

- Акт отбора и распределения проб готовой продукции
