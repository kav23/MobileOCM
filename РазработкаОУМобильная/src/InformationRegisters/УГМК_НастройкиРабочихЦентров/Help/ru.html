Настройка

В поле Настройка указывается:

"" - Разрешенные "Группы продукции" для материалов РЦ (Справочник.СК_ГруппыПродукции)

"Продукции" - Разрешенные  "Группы продукции" для продукции РЦ (Справочник.СК_ГруппыПродукции)

"Операции" - Доступные операции. (Справочник.УГМК_СостоянияЗаказа)

"РабочиеЦентры" - Доступные рабочие центры. (Справочник.РабочиеЦентры или Справочник.ГруппыЗаменяемостиРабочихЦентров)

"Режимы" - Доступные режимы. (Справочник.УГМК_РежимыРаботыРабочихЦентров)

"ТипПартии", "ТипПартииПродукции" - Тип БизнесПроцесса поиск и ввод которого производится у РЦ. (Строка "УГМК_ПаспортПартии", "УГМК_Плавка", "УГМК_ПартияСырья")

"МаскаВвода.УГМК_ПаспортПартии",  "МаскаВвода.УГМК_Плавка", "МаскаВвода.УГМК_ПартияСырья"- Маска ввода номера партии.

"ДопускаетИзменениеПартии" - Допустимо ли изменение Партии при обработке.

"Подразделения" - Доступные Подразделения. (Справочник.Подразделения)

"Склады" - Доступные Склады (Справочник.Склады)

"Пользователи" - Пользователи, доступные к выбору в диалоге Смены пользователя.

"Водители" - Сотрудники, которые могут быть выбраны в качестве водителя (Справочник.СотрудникиОрганизаций).

Приоритет

Поумолчанию значения приоритетов определяются по типу Объекта настройки.

-1 - общая настройка;

 0 - Пользователь и Рабочий центр;

10 - Группа заменяемости;

15 - Рабочее место;

20 - Подразделение.

Общие настройки доступны всегда (при любых параметрах вызова) Если по параметрам вызова обнаружена настройка с МЕНЬШИМ приоритетом, то настройки с большими значениями будут проигнорированы.

Например Если существуют настройки "Общие", "Подразделение" и для конкретного "Рабочего центра", то будут доступны "Общие"+"Рабочий центр"
